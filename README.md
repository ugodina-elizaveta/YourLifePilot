# YourLifePilot Bot 🤖

**YourLifePilot** — это Telegram-бот, который помогает наладить сон и чуть аккуратнее относиться к себе. Он не заменяет врача, а лишь напоминает о маленьких шагах, поддерживает и задаёт простые вопросы, чтобы пользователю было легче держать курс.

## 🌟 О проекте

YourLifePilot создан для тех, кто хочет улучшить качество сна и снизить уровень стресса. Бот проводит первичный онбординг, определяет сценарий пользователя и отправляет персонализированные уведомления с микро-задачами.

> ⚠️ **Важно:** Бот не оказывает медицинскую помощь и не заменяет консультацию врача или психолога. При серьёзных проблемах со здоровьем необходимо обратиться к специалисту.

## 🎯 Функциональность

### 1. Онбординг (5 вопросов)
- Оценка удовлетворённости сном за последние 2 недели
- Желаемое время отхода ко сну
- Фактическое время засыпания
- Самочувствие после пробуждения
- Время наибольшего стресса в течение дня

### 2. Определение сценария
На основе ответов пользователь относится к одному или нескольким типам:
- 😴 **"Ложусь поздно"**
- 🥱 **"Просыпаюсь разбитым"** 
- 😰 **"Днём высокий стресс"**

### 3. Типы уведомлений

#### 🌅 Утренние
- Чек-ин о качестве пробуждения
- Микро-задачи для бодрости (дыхательные упражнения)

#### 🌆 Дневные (только для сценария "высокий стресс")
- Короткие паузы для снижения стресса
- Дыхательные практики

#### 🌃 Вечерние
- Подготовка ко сну
- Вопрос о самочувствии с сохранением статистики

### 4. Адаптивная логика
- **3+ успешных выполнения подряд** → похвала и усиление задачи
- **3+ пропуска подряд** → упрощение задачи и мягкая поддержка
